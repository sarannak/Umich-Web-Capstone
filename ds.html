What is Data Structure?

Data Structure is a mathematical or logical model according to which data items are stored inside the computer memory. The purpose of a data structure is to organize data into memory so that it can be accessed quickly and efficiently. Example: lists,stacks,queues,trees,graphs,etc...

Array(Linear):

An array is a collection of the same type of data itmes, which are stored in consecutive memory locations under a common name. An array variable must be followed by square bracket enclosing size specification of the array. An Array element can be accessed as 'array_name[n]' , n is an integer. Size of the array has to be always specified before usage.

Linked List(Linear):

A Linked List is a collection of homogeneous data elements (called node), but not stored contagiously in the computer memory. The linear order is maintained by means of links or pointers. Unlike arrays their size are not specified and memory is allocated for each element separately. But access to each element in the list is not as simple as in arrays.

Stack(Linear):

A stack is one of the most important and useful non-primitive data structure in computer science. It is a sequential collection of elements into which new elements are inserted and taken out from only one end. Stack is also called LIFO(Last In First Out) data structure. In a stack only the topmost element can be accessed.

Queue(Linear):

The queue is also another useful non-primitive linear data structure in computer science. A queue is a collection of elements in which deletions take place at one end called front and insertions take place at another end called rear. Queue is also called FIFO(First In First Out) data structure.

Tree(Non-Linear):

Tree is a non-linear data structure that can be used to represent data items possessing a hierarchical relationship. A Tree may be defined as a finite set of nodes such that there is a specially designated node called root and the remaining nodes are partitioned into zero or more disjoint trees called subtrees of the root node. This webpage is stored as a tree inside the RAM.

Graph(Non-Linear):

A graph is non linear data structure which consists of set of objects and a set of edges connecting the objects. Consider 4 cities Kolkata,Delhi,Mumbai and Chennai. These can be considered as objects or vertex of a graph and the roadways and railway lines connecting them can be considered as edges. Graphs has important applications in networking and other areas of computer science.
