What is a Computer?

A computer is a machine that can be instructed to carry out sequences of arithmetic or logical operations automatically.Early computers were only conceived as calculating devices. A computer can be built using mechanical,electrical,etc.. components. Early in the Industrial Revolution, some mechanical devices were built to automate long tedious tasks. Nowadays computers are digital in nature i.e everything(files,texts, even images and videos) are represented as digits inside the computer. Today's computer uses binary number system i.e a system in which there are only 2 digits - 0 and 1 (can be represented as off or on using a switch).

Components of a Computer

There are 3 major components:
Processor/CPU: It is the brain of the computer which performs all the operations the computer. It has a component called ALU(Arithmetic Logic Unit) which perform all the arithmetic and logic operations of CPU. The operation of the ALU is controlled by CU(Control Unit).
Memory Unit: It stores the data/instructions of the computer.There are 2 types of memory, Primary(Volatile) and Secondary(Non Volatile) memory.
I/O Devices: These are used to connect the computer to the peripheral world. I/O is mainly used to provide some input to the computer and get some output.

What is Programming?

Programming is the process of giving instructions to do a specific job or solve a problem. The process of writing instructions in Languages understood by the computer is known as computer programming. Basically, a program is a set of instructions written in a particular sequence which can either be stored in a computer's memory or given as input using I/O devices. A collection of programs is called Software and the physical Components are called Hardware.
